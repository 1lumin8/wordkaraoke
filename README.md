# LyricSync Studio

A browser-based karaoke lyric sync tool.

## What it does

- Upload a song locally in the browser.
- Paste lyrics line by line.
- Tap along to sync each lyric line to the song.
- Save tap-sync progress in the browser for the same song and lyrics.
- Export synced lyrics as JSON or LRC.

## Files

- `index.html`
- `styles.css`
- `app.js`

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, `app.js`, and this `README.md`.
3. In the repo, go to Settings > Pages.
4. Under Build and deployment, choose Deploy from a branch.
5. Select the `main` branch and `/root`.
6. Save.

GitHub will give you a public website link after it deploys.

## Privacy

Songs are processed locally in the user's browser. The current version does not upload audio files to a server.
